## Contingency Triggers

**If Something Goes Wrong:**

### Website Down
→ Activate landing page backup
→ Post transparency update on Twitter
→ Provide ETA for restoration
→ Escalate to Ops immediately

### Major Negative Feedback
→ Flag to CEO immediately
→ Assess validity (is criticism fair or just noise?)
→ Prepare thoughtful response (don't react immediately)
→ Adjust messaging if needed for Day 2

### Low Response Numbers
→ Activate incentive boost (gift cards for respondents)
→ Email framework maintainers for amplification
→ Increase social media posting frequency
→ Consider paid advertising boost

### Messaging Conflict Detected
→ Halt that channel's posting immediately
→ Coordinate fix with relevant teams
→ Clarify correct messaging
→ Resume with corrected messaging
